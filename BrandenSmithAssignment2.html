My Favorite Tags

SVSU CIS 255

Introduction

The purpose of this document is to show and explain my top three favorite HyperText Markup Language (HTML) tags. I am a Computer Science student at Saginaw Valley State University doing a HTML assignment for CIS 255.

<h1> to <h6>. The header tag is one of my favorite tags. Although the header tag is very simple, I like it because it has six different sizes. This feature allows the designer to easily create a hierarchy of headers and formatting without compilicated styling, as you can see above.

<ul>. The unordered list tag is also a really cool tag.
Reasons:

  • Appealing way to show a list.
  • Bullets instead of numbers allow it to be unordered.
  • Lists can go on...
  • ... and on...
  • ...and on.

<a>. According to Dr.George Corser in his essay MY FAVORITE HTML TAGS, the anchor tag "most fundamentally embodies the purpose of the World Wide Web: to enable documents to be linked to other documents." I couldn't agree more. The Web needs links to help us travel from one place to another, and links need the anchor tag in order to work. Anchor tags and links are very important for websites like google.com to exist.

Conclusion

The header, unordered list, and anchor tags are three of my favorite tags because they are all very useful in their own ways. The header and unordered list tags are helpful when formatting or designing. While the anchor tag is a needed component for the Web to function the way it does.
